linux安装jdk

发表于 2018-03-10 | 分类于 java | 阅读次数

​ 这段时间找实习，上班，荒废了好久，周末撸起袖子，记录下今天的所写吧。今天主要写了下，怎么在linux中的安装jdk。我主要用的是红帽的linux 6版本，当初在学校安装的。就当试验下，之后在买个centos的与服务器。

1.查看下当前的linux环境下是否有java版本。删除当前的java版本

查看java ：java -version

查看rpm -qa | grep jdk

删除sudo yum remove java-1.6.0-openjdk-1.6.0.0-1.21.b17.el6.i686

2.下载新的jdk

到oracle官网下载jdk

命令： wget http://download.oracle.com/otn-pub/java/jdk/8u161-b12/2f38c3b165be4555a1fa6e98c45e0808/jdk-8u161-linux-x64.tar.gz?AuthParam=1520666889_774d1212ee1e2549fcbc4553be96841a

解压下载后的jdk包

sudo rpm -ivh jdk-7u80-linux-x64.rpm

3.cd /usr/java

cd jdk的安装目录

sudo vim /etc/profile export path=配置环境变量

source /etc/profile 生效

java -version
